1

Identify the coefficients in the standard form ax² + bx + c = 0:

a = 2, b = 5, c = -3

2

Apply the quadratic formula:

x = (-b ± √(b² - 4ac)) / (2a)

3

Calculate the discriminant:

b² - 4ac = 5² - 4(2)(-3) = 25 + 24 = 49

4

Substitute into the formula:

x = (-5 ± √49) / (2×2) = (-5 ± 7) / 4

Final Answer:

x₁ = (-5 + 7) / 4 = 1/2

x₂ = (-5 - 7) / 4 = -3
